Agricultural Yield Analysis using Machine Learning

Overview

This project performs Exploratory Data Analysis (EDA) and Machine Learning on an Agricultural Yield Dataset to understand the factors affecting crop yield and build a predictive model.

Dataset Features

- Crop_Type
- Soil_Type
- Region
- Season
- Irrigation
- Fertilizer
- Rainfall
- Temperature
- Humidity
- Nitrogen
- Yield

Tasks Performed

Dataset Understanding

- Dataset overview
- Data type analysis
- Missing value detection

Exploratory Data Analysis (EDA)

- Histograms and distribution analysis
- Crop type analysis
- Soil type analysis
- Scatter plot analysis
- Correlation analysis using heatmap
- Group-based yield analysis

Data Preparation

- Missing value handling
- One-Hot Encoding
- Feature selection
- Train-test split

Machine Learning

- Linear Regression Model
- Model coefficient analysis
- Yield prediction

Tools and Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

Objective

To analyze agricultural data, identify factors influencing crop yield, and apply machine learning techniques for yield prediction.
